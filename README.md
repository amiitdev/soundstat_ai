# 🎵 SoundStat AI - Intelligent Music Discovery Platform

![SoundStat AI](https://img.shields.io/badge/AI-Powered-1DB954?style=for-the-badge&logo=spotify)
![License](https://img.shields.io/badge/Open%20Source-MIT-000000?style=for-the-badge)
![Version](https://img.shields.io/badge/Version-1.0.0-FF416C?style=for-the-badge)

A modern, responsive music streaming web application with **AI-powered features** that revolutionize how you discover, analyze, and enjoy music. Built with vanilla JavaScript and powered by **Puter.js** for free, unlimited AI capabilities.

---

## ✨ Key Features

### 🎧 AI-Powered Music Discovery
- **Smart Genre Recommendations** - Click any mood/genre to get 20-25 AI-curated songs
- **Intelligent Search** - Voice search support with Web Speech API
- **AI Song Suggestions** - Automatic recommendations based on currently playing track
- **Personalized Greeting** - Daily music recommendations based on your listening history

### 🤖 Advanced AI Analysis
- **Lyrics Analysis** - AI analyzes song lyrics for themes, mood, and meaning
- **Artist Biography** - Automatic artist background and career highlights
- **Song Similarity Score** - Compare any two songs with AI-powered similarity rating (0-100%)
- **AI Sound Prompts** - Generate creative sound design ideas for music producers

### 📱 Playlist Generation
- **Mood-Based Playlists** - Generate 10-song playlists for Studying, Workout, Party, or Relaxation
- **One-Click Creation** - AI creates curated playlists instantly

### 🎨 Music Player
- **Fullscreen Player** with rotating album art
- **Audio Effects** - Bass Boost, Rock Mode, Cinema Reverb
- **Equalizer Visualization** - Real-time audio frequency display
- **Shuffle & Repeat** modes
- **Queue Management** - Add songs to favorites/liked songs
- **Sleep Timer** ready (infrastructure in place)

### 📱 Fully Responsive
- **Mobile-First Design** - Optimized for all screen sizes
- **Hamburger Menu** - Genre sidebar for mobile devices
- **Touch-Friendly** - Large buttons and gesture support
- **Bottom Navigation** - Easy thumb access on mobile

## 🚀 How to Use

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/soundstat-ai.git
cd soundstat-ai
```

### 2. Open in Browser
Simply open `index.html` in any modern browser:
- Chrome/Edge (recommended for voice search)
- Firefox
- Safari

No build step, no npm install, no server required!

### 3. Start Exploring
- **Search** - Type or speak any song/artist
- **Genres** - Click mood chips or use hamburger menu (mobile)
- **Play** - Click any song to start playing
- **AI Features** - Try lyrics analysis, artist bio, or playlist generator

## 🌟 AI Features Architecture

```
User Action → Puter.js AI → Gemini 2.5 Flash → Response
```

**Example: Mood Playlist Generation**
1. User clicks "Rock" genre
2. Puter.js sends request to Gemini 2.5 Flash
3. AI returns: "Bohemian Rhapsody - Queen, Stairway to Heaven - Led Zeppelin..."
4. App searches each song on Saavn API
5. Displays 20-25 curated tracks

## ⚖️ Troubleshooting

### Voice Search Not Working
- Use Chrome or Edge (Firefox doesn't support Web Speech API)
- Check browser console for errors (F12)
- Ensure microphone permissions are granted

### Songs Not Loading
- Check internet connection
- Saavn API might be rate-limited (wait a few seconds)
- Check browser console for API errors

### AI Features Not Responding
- Puter.js requires internet connection
- No API key needed - works out of the box
- Check if `https://js.puter.com/v2/` is not blocked
